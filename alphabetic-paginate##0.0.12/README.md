# Alphabetic Paginate

Paginates in alphabetic groups

## Installation

Add this line to your application's Gemfile:

    gem 'alphabetic_paginate'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install alphabetic_paginate
