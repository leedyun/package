

##Install

Then, just add this to your Gemfile:

`gem 'activerecord-db-tools'`

And run your bundler:

`bundle install`

##Usage

If you want to use it just put in your Gemfile:

    gem 'activerecord-db-tools'

Then to turn on set the environment variable:

    DB_READ_ONLY=true
