# Capistrano::Drupal

Drupal specific tasks for Capistrano v3

## Installation

```
# Gemfile
gem 'capistrano',  '~> 3.1'
gem 'capistrano3-drupal', '~> 0.1', :github => 'gregoiredavid/capistrano-drupal'
```

## Usage

Require capistrano3-drupal in your cap file

```
# Capfile
require 'capistrano/drupal'
```
