# Moving Words

Moving Words is a parser for video caption file formats. It currently supports SubRip, or SRT.

## Installation

Add this line to your application's Gemfile:

    gem 'moving_words'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install moving_words
